# Singular Solutions — Landing Page (Next.js)

Landing page do grupo **Singular Solutions — Desenvolvedores Associados**.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra http://localhost:3000

## Build de produção

```bash
npm run build
npm start
```

## Stack
- Next.js 14 (App Router) + TypeScript
- three.js (fundo animado `MagicRings`)
- Fonte Graduate (Google Fonts via next/font)
- Estilos em `app/globals.css` (paleta teal/dark)
- Bilingue PT/EN (toggle no menu)
